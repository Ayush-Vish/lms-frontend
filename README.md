# Learning management System frontEnd 

### Setup 

1.Clone the project 

```bash
    git clone https://github.com/Ayush-Vish/lms-frontend.git
```
2. Move into the directory

```bash
    cd lms-frontend
```
3. Install dependencies

```bash
    npm i 
```

4. Run the server 
```bash
    npm run dev 
```
